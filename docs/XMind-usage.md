## XMind快捷键

| 项目         | 快捷键       |
| ------------ | ------------ |
| 子主题       | Tab          |
| 主题（之后） | Enter        |
| 主题（之前） | Shift+Enter  |
| 父主题       | Ctrl+Enter   |
| 联系         | Ctrl+Shift+L |
| 外框         | Ctrl+Shift+B |
| 概要         | Ctrl+Shift+] |
| 笔记         | Ctrl+Shift+N |
| 超链接       | Ctrl+K       |
| 优先级       | Ctrl+1~7     |
