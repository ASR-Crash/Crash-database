You-Get使用

安装

    pip3 install you-get

升级

    pip3 install --upgrade you-get

查看格式

    you-get -i [url]

下载

默认输出至user文件夹：

    you-get [url]

指定路径：

    you-get -o [path] -O [filename] [url]

暂停\重新下载

    ctrl+c


    you-get -f [url]
