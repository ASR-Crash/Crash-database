## PowerToys Usage

### 始终置顶

		Win+Ctrl+T

### Color Picker

		Win+Shift+C

### FancyZones

		Win+`

### PowerToys Run

		Alt+Space

### 屏幕标尺

		Win+Shift+M

### Shortcut Guide

		Win+Shift+/

### 文本提取器

		Win+Shift+T
