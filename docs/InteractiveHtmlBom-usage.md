# InteractiveHtmlBom使用

- AD另存为 PCB ASCII File
- 立创EDA导入 PCB ASCII File
- 立创EDA导出 .json
- json文件拖入 InteractiveHtmlBom_python_inside\Run.bat
- 点击 Generate BOM
- Done！

---

## Comment值改为Value

- 原理图选中需要更改的器件
- Find Similar Objects -> Description -> Same
- Ctrl+A -> Properties -> Comment -> ··· 
- Smart Edit -> Formula -> =Value
